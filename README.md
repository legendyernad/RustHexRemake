# RustHexRemake

RustHex Remake — проект, нацеленный на перенос игры Rust в Minecraft! На данный момент проект находится на стадии закрытого бета тестирования, подробности в дискорд сервере https://discord.gg/gp4PrYc

Development using: Spigot 1.16.5, MCP-REBORN 1.16.5, OptiFine G5, lwjgl 3, rust assets

Developers: YernaD, others
